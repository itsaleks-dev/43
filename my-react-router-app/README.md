##### React Router Demo App #####

## Dependencies
`react`
`react-dom`
`react-router-dom`

`vite`
`@vitejs/plugin-react`

## Installation and launch
Clone the repository: https://github.com/itsaleks-dev/43
cd my-react-router-app
npm install
npm run dev

## Build and preview
npm run build
npm run preview

## Deployment
Vercel: https://43-beta.vercel.app
